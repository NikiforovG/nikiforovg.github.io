About the Project

For one of my personal projects, I needed to deploy ClickHouse on Google Cloud Platform (GCP). To embrace the Infrastructure as Code (IaC) approach, I opted to use Terraform. The repository for this project contains Terraform scripts that automate the creation of a virtual machine (VM) on Google Compute Engine. It also sets up an additional persistent SSD, specifically for data storage, and configures the network to allow traffic through the ClickHouse port. Additionally, I assigned a static IP to the VM for consistent access.
